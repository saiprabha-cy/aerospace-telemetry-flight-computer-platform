# Ground Station

## Overview

The Ground Station receives telemetry packets from the Telemetry Engine and displays mission information to the operator.

## Responsibilities

- Receive telemetry packets
- Decode telemetry
- Display mission information
- Generate mission report

## Input

telemetry_packets.json

## Output

mission_report.txt
