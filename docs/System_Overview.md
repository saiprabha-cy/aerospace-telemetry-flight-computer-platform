# System Overview

## Project Name

Aerospace Telemetry and Flight Computer Integrated Development Platform (ATFIDP)

---

## Purpose

ATFIDP is a software-first aerospace avionics development platform that simulates the major software subsystems commonly found in launch vehicles, sounding rockets, reusable rockets, CubeSats, and spacecraft.

Instead of beginning with expensive hardware, the platform focuses on validating software architecture, telemetry pipelines, flight computer logic, and health monitoring before deployment to embedded hardware.

---

## Goals

- Learn aerospace embedded software development
- Simulate onboard flight computer behavior
- Generate virtual sensor data
- Build a telemetry communication pipeline
- Monitor mission health
- Prepare software for STM32 deployment
- Develop a reusable aerospace software architecture
