# Sensor Specification

## Temperature Sensor

Purpose:
Monitor onboard temperature.

Range:
-40°C to 120°C

Update Rate:
1 Hz

---

## Altitude Sensor

Purpose:
Estimate vehicle altitude.

Range:
0m to 10000m

Update Rate:
10 Hz

---

## Battery Voltage Sensor

Purpose:
Monitor power system health.

Range:
0V to 24V

Update Rate:
1 Hz

---

## Velocity Sensor

Purpose:
Estimate vehicle velocity.

Range:
0 to 1000 m/s

Update Rate:
10 Hz
