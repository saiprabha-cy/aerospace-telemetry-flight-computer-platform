# Aerospace Telemetry and Flight Computer Integrated Development Platform

A software-first aerospace avionics simulation platform.

Features:

- Virtual Sensors
- Flight Computer
- Telemetry Engine
- Ground Station
- Fault Detection
- Launch Readiness System

Technology Stack:

- Python
- STM32CubeIDE
- GNU Octave
- Git/GitHub
- Draw.io
